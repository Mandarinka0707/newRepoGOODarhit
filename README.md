# Форум для обсуждений 

Полноценная веб-платформа для общения, включающая систему публикаций, комментариев, интерактивный чат, а также защищенную регистрацию и аутентификацию пользователей. Проект спроектирован с нуля, содержит задокументированное REST API и высокий уровень покрытия автоматическими тестами.

## Технологический стек

* **Backend:** Go (Golang)
* **Frontend:** JavaScript, React, HTML5, CSS3
* **База данных:** PostgreSQL, PL/pgSQL
* **Документация API:** Swagger (OpenAPI)
* **Тестирование:** Go testing (Unit & Integration tests)

##  Основной функционал

* **Пользователи:** Регистрация, безопасная аутентификация и авторизация сессий.
* **Контент:** Создание тематических постов (публикаций) и возможность оставлять комментарии под ними.
* **Интерактив:** Реализация чата для обмена сообщениями в реальном времени.
* **Документация:** Полная спецификация всех эндпоинтов в Swagger для удобного тестирования API.

## Тестирование и обеспечение качества (QA)

Проекту уделено особое внимание с точки зрения качества кода и надежности системы:
* Написаны комплексные **юнит- тесты** для проверки изолированной бизнес-логики модулей.
* Реализованы **интеграционные тесты** для проверки корректности взаимодействия бэкенда с базой данных и API.
* Общее покрытие кода тестами (**Code Coverage**) составляет **более 80%**.

## Структура проекта (кратко)

* `/backend` — Исходный код серверной части на Go, включая миграции БД, спецификацию Swagger и автотесты.
* `/frontend` — Клиентское приложение на React (JS/HTML/CSS).
